The Drift

Visual elements travel slower than the page itself, so they appear suspended in space — like objects seen from a moving train window.

02

The Counter-drift

This orb moves against the scroll direction. Opposing motion between layers is what tricks the eye into perceiving depth.

The Glow

Soft radial gradients and blur create atmosphere without a single image file — this entire site ships with zero raster assets.
